- Threat libraries, research portals, and multilateral initiatives

### How to use this directory
- Expand a category to browse the relevant frameworks or authorities.
- Select any entry to open the official external website.
- Use this module for quick cross-references, verification, or context when reviewing
  CRT controls, mappings, or scenarios.

### Troubleshooting
- If no entries are shown, verify that `docs/catalogues/cyber_references.yaml` exists and is correctly structured.  
- To add new items, edit the YAML catalogue with the same schema.  
- If an external link fails, the provider may have updated its URL.

### Notes
